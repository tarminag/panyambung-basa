# CHANGELOG — Panyambung Basa
All notable changes to this project will be documented in this file.

The format follows **Keep a Changelog**, and this project adheres to
**Semantic Versioning** principles, adapted for a curated linguistic project.

---

## [v1.1.0] — In Progress
### Phase: Refinement & Curation Readiness

#### Added
- CONTRIBUTING.md (trilingual: Sunda–Indonesia–English)
- ROADMAP.md (trilingual)
- ROADMAP_v1.1.md (focused roadmap for refinement phase)
- GitHub Issue Template for term proposals (trilingual)
- Pull Request Template with curation checklist
- Initial workflow documentation for curated contributions

#### Changed
- Clarified project philosophy: **Open Use – Curated Source**
- Improved documentation structure and consistency
- Minor refinements in terminology metadata (non-breaking)

#### Deprecated
- Informal contribution via direct commits without review
- Unstructured terminology changes without references

#### Removed
- None

#### Fixed
- Documentation gaps related to contribution workflow
- Ambiguity in curator authority description
